# 암호화 게시판 서비스 


## 기술 스택

- Framework: `NestJS`
- Database: `RDS - mysql`
- ORM: `TypeORM`


# 요구사항 분석

## 1. 게시글 등록
  - 유저가 게시글을 등록 합니다.
  - 제목은 최대 20자, 본문은 200자 입니다.
  - 비밀번호를 설정 가능하며 최소 6자이고 숫자 1개 이상을 포함해야 합니다.
   
## 2. 게시글 목록 조회
  - 게시글의 전체 목록을 조회 합니다.
  - 목록은 등록된 게시글의 최신 순으로 조회를 합니다.
  
## 3. 게시글 삭제
  - 게시글을 삭제합니다.
  - 게시글 생성시 입렵한 비밀번호와 동일해야 삭제가 가능합니다.
  
## 4. 게시들 수정
  - 게시글을 수정합니다.
  - 게시글 수정도 비밀번호를 알아야 수정이 가능합니다.
  - 제목과 내용을 수정할 수 있습니다.
